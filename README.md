PsLE Scaler
===========
 
FireworksでPhotoshopライブエフェクト（以下PsLE）を拡大縮小するコマンドです。
 
## 使い方
 
1. PsLEが適用されたオブジェクトを選択します。（複数選択可）
2. コマンドを実行します。
3. ダイヤログが表示されるので、拡大縮小率を入力します（単位は%）。数字の後ろに「/」（半角スラッシュ）を入力した場合、PsLEのみを拡大縮小します。入力例：200/
4. PsLEおよびオブジェクトが変形されます。

## 使用上の注意点
 
オブジェクトの変形は、「数値を入力して拡大・縮小」と同等の機能を使っているため、
 
- テキストオブジェクトに対して実行すると、テキストが扁平します。位置はずれますが「変形のリセット」をするか、こちらのコマンドを使ってください。[fix_transfromation_text.jsf](https://github.com/littlebusters/FwExtensions/blob/master/fix_transformation_text.jsf)
- 拡大縮小率やオブジェクトの大きさ・座標により、ハーフピクセルのできる場合があります。ピクセルにスナップで修正してください。
 
にご注意ください。
 
その他の仕様や既知のバグ等は以下の通りです。

- 幅・高さとも同じ比率での拡大縮小になります。
- スクリプトでPsLEが適用されているオブジェクトを走査しているため、選択しているオブジェクトが多い・グループ化のネストが深い・多数のフィルタを適用しているオブジェクトが多い場合など、Fireworks自体が落ちる可能性があります。
- PsLE（ドロップシャドウ・ストローク）のみを適用した50個のオブジェクトに対して実行した場合、約1.5秒かかります。（iMac(27-inch, Mid 2010) Core i7 / mem 16GB / HDD）
- パッケージ版CS6（Mac / Win）でテストをしています。
